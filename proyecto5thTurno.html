5th Turno v2.6

Calendario perpétuo para el 5º turno de trabajo de una compañía alimentaria.

Cómo se hizo

El objetivo principal de este proyecto era crear un calendario perpétuo para el 5º turno, facilitando la consulta de turnos y descansos de forma visual y sencilla.

Se desarrolló una interfaz intuitiva y responsiva, optimizada para dispositivos móviles y escritorio, con lógica de cálculo de turnos y festivos adaptable a cualquier año.

  • - HTML
  • - CSS

Proceso de Diseño (UI/UX)

El diseño se centró en la simplicidad y claridad visual, usando una paleta de colores coherente con el portfolio y componentes reutilizables. Se priorizó la experiencia móvil y la facilidad de consulta.

Retos y Soluciones

Cálculo de turnos

El mayor reto fue crear un algoritmo flexible para calcular los turnos y descansos de cualquier año.

Solución: Se implementó una lógica adaptable y validada con usuarios reales.

Accesibilidad y usabilidad

Garantizar que cualquier usuario pudiera consultar el calendario fácilmente desde cualquier dispositivo.

Solución: Se diseñó una interfaz clara, con contraste y navegación sencilla.
