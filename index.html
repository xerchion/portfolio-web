Sobre mí

Soy un desarrollador de software con una sólida formación en arquitectura y diseño de sistemas de calidad, reusables y escalables. Me apasiona la mejora continua y disfruto abordando el ciclo completo del producto, desde el análisis de requisitos hasta el despliegue. Cuento con una experiencia versátil, habiendo desarrollado proyectos en diversos lenguajes y tecnologías.

Mi stack técnico incluye lenguajes como Python, Java y JavaScript con frameworks como Django y Flask. Tengo experiencia con bases de datos como PostgreSQL, MySQL y MongoDB. Estoy familiarizado con metodologías Agile y herramientas de gestión de proyectos como Git y GitHub.

Mi objetivo es aplicar mis habilidades para construir soluciones robustas y eficientes que generen un impacto positivo. Siempre estoy buscando nuevos retos que me permitan crecer tanto a nivel personal como profesional.
